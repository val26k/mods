># <b># NotificationsOverhaul</b>

Overhauls the boring notification system with a configurable one.

</br>

![](https://i.imgur.com/fERmuXa.png)

</br>

>## ## Installation (manual)

- extract `NotificationsOverhaul.dll` file to your `BepInEx\Plugins\` folder

</br>

>## ## Features

- Configurable Duration, MaxVisibleNotifications and Position.
- Configurable FontSize and Color.
- Can be configured to only apply to pickup notifications (picking up items).
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
