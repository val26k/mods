># <b># GuckCollector</b>

Adds a new build piece that collects guck from infested trees.
<br/>

![](https://i.ibb.co/Ln479XS/ss2.png)

>## ## Installation (manual)

- extract `GuckCollector.dll` file to your `BepInEx\Plugins\` folder

<br/>

>## ## Features

- Adds a new build piece called GuckCollector.
- Works in different biomes as long as its connected to a guck sack.
- Configurable duration, max capacity and recipe.
- Uses serversync and built-in config watcher.
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
