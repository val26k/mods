# Crafting Station Tweakz

A small mod that gives you more control over how crafting stations and their upgrades work. Customize things like build range, upgrade connection distance, spacing requirements between upgrades, and more!

The inspiration for this mod came from the spacing requirement between upgrades, something that has always annoyed me a little. I don’t like hiding upgrades underground, and honestly, they make great decorations. One of the main goals of this mod is to remove those placement restrictions, allowing you to place upgrades wherever you want, including purely for decoration.

## Config
These config values apply to **all** crafting stations:
- **Build Range**: The base build range of the crafting station. Default: **20**
- **Build Range Increase per Upgrade**: Each upgrade increases the crafting station's build range. Default: **4**
- **Upgrade Range**: The maximum distance at which upgrades still connect to the crafting station. Default: **false**
- **Space Requirement Between Upgrades**: The minimum required spacing between upgrades in order to place them (game default is 2). Default: **0**
