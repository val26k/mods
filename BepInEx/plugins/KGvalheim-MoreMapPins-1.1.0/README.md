Mod allows you to add map pins to place on map. Up to 250 new pins. Icons should be placed in BepInEx/config/MoreMapPins/ folder with .png format and named as integer value. Keep in mind that ORDER OF FILES (Their filename number) MATTERS. If you will in future change icon name and order will shift it will also shift icon ID ingame, so it'll be different one.
