# <b># BetterStations</b>

A mod inspired by Enshroudeds crafting stations. Overhauls how crating/processing stations work, instead of interacting and pressing E key everytime you need to add ore/fuel, just put everything on its storage bin and it will do its job for you.

>## ## Installation (manual)

- extract `BetterStations.dll` file to your `BepInEx\Plugins\` folder.

<br/>

>## ## Features

- Overhauls the smelter/blastfurnace, kiln, spinning wheel, eitr refinery and windmill (will add more in the future (ie. fermenter etc..))
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
