># <b># FeatherCollector</b>

Adds a new build piece that collects feathers if you build it on a tree.
<br/>

![](https://i.ibb.co/Kht1yrt/ss3.png)

>## ## Installation (manual)

- extract `FeatherCollector.dll` file to your `BepInEx\Plugins\` folder

<br/>

>## ## Features

- Adds a new build piece called FeatherCollector, only works if you build it on a tree.
- Only works in the meadows, black forest and plains.
- Doesn't work if it rains, or not properly connected to a tree.
- Configurable duration, max capacity and recipe.
- Uses serversync and built-in config watcher.
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
