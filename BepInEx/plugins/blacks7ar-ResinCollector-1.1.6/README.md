># <b># ResinCollector</b>

Requested by a NexusMod user, adds a new build piece that collects resins from trees.
<br/>

![](https://i.ibb.co/2t2XyB2/ss.png)

>## ## Installation (manual)

- extract `ResinCollector.dll` file to your `BepInEx\Plugins\` folder

<br/>

>## ## Features

- Adds a new build piece called ResinCollector, only works if you build it on a tree.
- Only works in the meadows, black forest and plains.
- Doesn't work if not properly connected to a tree.
- Configurable duration, max capacity and recipe.
- Uses serversync and built-in config watcher.
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
