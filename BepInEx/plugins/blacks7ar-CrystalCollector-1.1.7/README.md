># <b># CrystalCollector</b>

Requested by a Nexusmod user, adds a build piece that collects cold air and turns it into crystal ice.
<br/>

![](https://i.ibb.co/NLcZNcf/ss1.png)

>## ## Installation (manual)

- extract `CrystalCollector.dll` file to your `BepInEx\Plugins\` folder

<br/>

>## ## Features

- Adds a new build piece called CrystalCollector, only works in cold and snowy areas.
- Stops working if theres a SnowStorm.
- Configurable duration, max capacity and recipe.
- Uses serversync and built-in config watcher.
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
