# <b># MagicBows</b>

Adds 10 end-game magical bows that uses eitr and doesn't need an arrow.

<br/>

![](https://i.ibb.co/dkg2vnw/collage1.png)
![](https://i.ibb.co/8DyS18M/front2.png)

<br/>

>## ## Installation (manual)

- extract `MagicBows.dll` file to your `BepInEx\Plugins` folder.

<br/>

>## ## Features

- Adds 5 Magical Bows and 5 Magical Crossbows.
- Configurable elemental damage, pierce damage, attack force and eitr drain.
- Uses ServerSync and built in config watcher.
- Configurable through config file or [Configuration Manager](https://valheim.thunderstore.io/package/Azumatt/Official_BepInEx_ConfigurationManager/)
